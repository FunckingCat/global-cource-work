# global-cource-work

## Тема - 	Скидки и спецпредложения

## Задание

- многостраничный сайт с БД 
(min 4 страницы, min 6 таблиц, реализован CRUD, вывод агрегирующей информации на странице)
- настройка административного интерфейса Django
